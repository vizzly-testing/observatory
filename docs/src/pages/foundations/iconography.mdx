# Iconography

Observatory uses [Heroicons](https://heroicons.com) — a beautiful, hand-crafted icon set designed by the makers of Tailwind CSS. Consistent icon usage improves scannability and reduces cognitive load.

## Installation

```bash
npm install @heroicons/react
```

## Import Pattern

```jsx
// Outline style (default)
import { CheckCircleIcon } from '@heroicons/react/24/outline';

// Solid style (for emphasis)
import { CheckCircleIcon } from '@heroicons/react/24/solid';

// Mini style (16px)
import { CheckCircleIcon } from '@heroicons/react/20/solid';
```

## Sizes

Use consistent sizes based on context.

  <IconSizesDemo />
</ComponentPreview>

```jsx
{/* Inline with text */}
<Icon className="w-4 h-4" />

{/* Buttons, form inputs */}
<Icon className="w-5 h-5" />

{/* Standard UI elements */}
<Icon className="w-6 h-6" />

{/* Cards, feature highlights */}
<Icon className="w-8 h-8" />

{/* Empty states, heroes */}
<Icon className="w-12 h-12" />
```

## Best Practices

### Do

- Use `shrink-0` on icons in flex containers to prevent squishing
- Match icon size to text size (w-4 with text-sm, w-5 with text-base)
- Use semantic colors consistently (green = success, red = error)
- Add hover states to interactive icons
- Include `aria-label` on icon-only buttons

### Don't

- Don't use icons without purpose—every icon should convey meaning
- Don't mix icon styles (outline + solid) in the same context
- Don't use oversized icons in dense UI
- Don't rely on icons alone—pair with text for accessibility

## Animation

Add subtle animation for processing states:

```jsx
{/* Spinning for loading */}
<ArrowPathIcon className="w-5 h-5 animate-spin" />

{/* Pulse for attention */}
<BellIcon className="w-5 h-5 animate-pulse" />
```

## Accessibility

Always provide context for screen readers:

```jsx
{/* Icon with visible label - label provides context */}
<div className="flex items-center gap-2">
  <CheckCircleIcon className="w-5 h-5" aria-hidden="true" />
  <span>Approved</span>
</div>

{/* Icon-only button - needs aria-label */}
<button aria-label="Close dialog">
  <XMarkIcon className="w-5 h-5" />
</button>

{/* Decorative icons */}
<Icon aria-hidden="true" />
```
