# Textarea

Textarea provides multi-line text input for longer content like descriptions, comments, and notes.

## When to use

- **Descriptions** — Project descriptions, build notes
- **Comments** — Review comments, feedback
- **Rich content** — When content may span multiple lines
- **Freeform text** — Notes, messages, documentation

## Basic Usage

<ComponentPreview client:load>
  <TextareaBasicDemo />
</ComponentPreview>

```jsx
<Textarea
  label="Description"
  placeholder="Describe your project..."
  hint="Markdown is supported"
/>
```
