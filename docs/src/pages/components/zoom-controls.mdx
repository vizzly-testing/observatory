# ZoomControls

ZoomControls provide buttons and presets for controlling image zoom level in the screenshot viewer.

## When to use

- **Image viewer** — Control zoom in screenshot comparison
- **Detail inspection** — Zoom into specific areas
- **Fit-to-screen** — Reset to optimal viewing size

## Desktop Variant

<ComponentPreview client:load>
  <ZoomControlsBasicDemo />
</ComponentPreview>

```jsx
let [zoom, setZoom] = useState('fit');

<ZoomControls zoom={zoom} onZoomChange={setZoom} />
```


## Zoom Levels

| Value | Description |
|-------|-------------|
| `'fit'` | Fit image to container |
| `'100'` | 100% actual size |
| `'50'` | 50% zoom |
| `'200'` | 200% zoom |
| Number | Custom percentage |

## Keyboard Shortcuts

Desktop variant supports keyboard shortcuts:

| Key | Action |
|-----|--------|
| `+` / `=` | Zoom in |
| `-` | Zoom out |
| `0` | Reset to 100% |
| `9` | Fit to screen |

## Best Practices

<DosAndDonts client:load>
  <Do>Show the current zoom level so users know where they are</Do>
  <Dont>Let users zoom so far they lose context</Dont>
</DosAndDonts>

<DosAndDonts client:load>
  <Do>Provide quick access to common zoom levels (fit, 100%)</Do>
  <Dont>Require many clicks to reach common zoom levels</Dont>
</DosAndDonts>
