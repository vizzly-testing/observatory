# FilterPill

FilterPills provide quick toggle filters for lists and tables. They show counts and can be combined in groups.

## When to use

- **Status filtering** — Filter by approved, pending, rejected
- **Category filtering** — Filter by type, platform, branch
- **Quick toggles** — Enable/disable filter groups
- **Multi-select** — When users can select multiple filters

## Basic Usage

<ComponentPreview client:load>
  <FilterPillBasicDemo />
</ComponentPreview>

```jsx
<FilterPillGroup>
  <FilterPill label="All" count={42} color="gray" active={!hasFilters} onClick={clearFilters} />
  <FilterDivider />
  <FilterPill label="Changed" count={8} color="amber" active={isActive('changed')} onClick={toggle} />
  <FilterPill label="New" count={5} color="cyan" active={isActive('new')} onClick={toggle} />
  <FilterPill label="Approved" count={29} color="emerald" active={isActive('approved')} onClick={toggle} />
</FilterPillGroup>
```

## Color Variants

<ComponentPreview client:load>
  <FilterPillColorsDemo />
</ComponentPreview>

| Color | Usage |
|-------|-------|
| `amber` | Changed, pending, warning states |
| `cyan` | New, added items |
| `emerald` / `green` | Approved, success states |
| `red` | Rejected, errors |
| `blue` | Info, branches |
| `purple` | Special categories |
| `gray` | Default, "all" option |

## Components

### FilterPillGroup

Wrapper that provides consistent spacing and layout.

### FilterPill

Individual filter toggle with label, count, and active state.

### FilterDivider

Vertical separator between logical groups of pills.

## Best Practices

<DosAndDonts client:load>
  <Do>Show counts to help users understand filter impact</Do>
  <Dont>Hide counts if they would always be zero</Dont>

<DosAndDonts client:load>
