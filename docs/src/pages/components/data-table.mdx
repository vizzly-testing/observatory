# DataTable

DataTable provides advanced table features like sorting, column visibility, search, and mobile-responsive card layouts.

## When to use

- **Large datasets** — Many rows that need filtering/sorting
- **User customization** — Column visibility preferences
- **Complex data** — Multiple interactive features needed
- **Responsive needs** — Mobile card layout support

## Basic Usage

<ComponentPreview client:load>
  <DataTableBasicDemo />
</ComponentPreview>

```jsx
let [sort, setSort] = useState({ key: 'name', order: 'asc' });
let [columnVisibility, setColumnVisibility] = useState({
  name: true, status: true, platform: true
});

<DataTable
  data={data}
  columns={[
    { key: 'name', label: 'Name', sortable: true, priority: 'high' },
    { key: 'status', label: 'Status', sortable: true },
    { key: 'platform', label: 'Platform' },
  ]}
  sortBy={sort.key}
  sortOrder={sort.order}
  onSortChange={(key, order) => setSort({ key, order })}
  columnVisibility={columnVisibility}
  onColumnVisibilityChange={setColumnVisibility}
  toolbar={<SearchInput ... />}
  footer={<span>{data.length} items</span>}
/>
```

## Features

### Sorting

Click column headers to sort. Sortable columns show sort indicators.

### Column Visibility

Users can toggle columns on/off. Set `priority: 'high'` for columns that should always be visible on mobile.

### Mobile Card Layout

On small screens, DataTable automatically switches to a card layout. High-priority columns remain visible.

### Toolbar & Footer

Use the `toolbar` slot for search and filters. The `footer` slot is great for counts and pagination.
