# SearchInput

A specialized input for search and filtering with a built-in search icon and clear button.

## When to use

- **Filtering lists** — Search within tables, lists, or grids
- **Quick lookup** — Find specific items by name or keyword
- **Live search** — Real-time filtering as user types
- **Command palette** — Quick action search

## Basic Usage

<ComponentPreview client:load>
  <SearchInputBasicDemo />
</ComponentPreview>

```jsx
let [search, setSearch] = useState('');

<SearchInput
  value={search}
  onChange={setSearch}
  placeholder="Search screenshots..."
/>
```

## Features

- **Search icon** — Visual indicator of search functionality
- **Clear button** — Appears when there's content to clear
- **Controlled** — Value managed through props for predictable state

## Best Practices

<DosAndDonts client:load>
  <Do>Provide instant feedback as users type (filter results live)</Do>
  <Dont>Require users to press Enter before showing results</Dont>
</DosAndDonts>

<DosAndDonts client:load>
  <Do>Use descriptive placeholder text ("Search screenshots...")</Do>
  <Dont>Use generic placeholders like "Type here..."</Dont>
</DosAndDonts>
