# StatusDot

A minimal circular indicator that shows status at a glance. Use when you need a compact visual cue without text.

## When to use

- **Inline status** — Next to names, labels, or in tight spaces
- **Table cells** — When Badge would be too prominent
- **Lists** — Quick visual scanning of item states
- **Connection status** — Online/offline indicators

## Basic Usage

<ComponentPreview client:load>
  <StatusDotBasicDemo />
</ComponentPreview>

```jsx
<StatusDot status="success" />
<StatusDot status="warning" />
<StatusDot status="danger" />
<StatusDot status="processing" />
```

## Status Reference

| Status | Color | Usage |
|--------|-------|-------|
| `success` | Green | Approved, passing, online |
| `warning` | Amber | Pending, review needed |
| `danger` | Red | Failed, rejected, error |
| `processing` | Blue (animated) | In progress, loading |

## Best Practices

<DosAndDonts client:load>
  <Do>Use StatusDot for quick visual scanning in dense interfaces</Do>
  <Dont>Use StatusDot alone for critical status — add text for accessibility</Dont>
</DosAndDonts>

<DosAndDonts client:load>
  <Do>Pair with nearby text that explains the status</Do>
  <Dont>Use more than one StatusDot per item — it creates confusion</Dont>
</DosAndDonts>
