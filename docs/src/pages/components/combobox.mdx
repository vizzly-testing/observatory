# Combobox

Combobox combines a text input with a dropdown, allowing users to search and filter options as they type.

## When to use

- **Large option sets** — When there are many options to choose from
- **Searchable lists** — Branches, users, projects
- **Dynamic options** — Options loaded based on input
- **Typeahead** — Auto-complete as user types

## Basic Usage

<ComponentPreview client:load>
  <ComboboxBasicDemo />
</ComponentPreview>

```jsx
let [selected, setSelected] = useState(null);

let branches = [
  { id: 1, name: 'main' },
  { id: 2, name: 'develop' },
  { id: 3, name: 'feature/auth' },
];

<Combobox
  items={branches}
  itemToString={(item) => item?.name || ''}
  renderItem={(item, { selected }) => (
    <div className="flex justify-between">
      <span>{item.name}</span>
      {selected && <CheckIcon />}
    </div>
  )}
  selectedItem={selected}
  onSelectedItemChange={setSelected}
  label="Select Branch"
  placeholder="Search branches..."
/>
```

## Combobox vs Select

| Use Combobox when... | Use Select when... |
|---------------------|-------------------|
| 15+ options | Few options (3-15) |
| Users need to search | All options visible at once |
| Dynamic/async options | Static option list |
| Custom option rendering | Simple text options |
