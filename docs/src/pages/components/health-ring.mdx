# HealthRing

HealthRing displays a circular progress indicator, perfect for showing test suite health, approval rates, or other percentage metrics.

## When to use

- **Test health** — Overall pass rate of screenshots
- **Progress metrics** — Completion percentage
- **Approval rates** — Percentage of approved builds
- **Performance scores** — Quality or health indicators

## Basic Usage

<ComponentPreview client:load>
  <HealthRingBasicDemo />
</ComponentPreview>

```jsx
<HealthRing value={95} label="Health" />
<HealthRing value={72} label="Health" />
<HealthRing value={45} label="Health" />
```


## Color Thresholds

The ring color changes based on the value:

| Range | Color | Meaning |
|-------|-------|---------|
| 80-100 | Green | Healthy, good |
| 50-79 | Amber | Moderate, needs attention |
| 0-49 | Red | Poor, needs improvement |

## Best Practices

<DosAndDonts client:load>
  <Do>Use HealthRing for at-a-glance metrics that users scan quickly</Do>
  <Dont>Use for metrics that need precise values displayed prominently</Dont>
</DosAndDonts>

<DosAndDonts client:load>
  <Do>Include a label to explain what the metric represents</Do>
  <Dont>Show multiple HealthRings without labels or context</Dont>
</DosAndDonts>
