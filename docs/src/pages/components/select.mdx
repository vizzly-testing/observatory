# Select

Select provides a dropdown for choosing from predefined options. Use when the list of options is known and limited.

## When to use

- **Predefined choices** — Status, category, priority
- **Limited options** — When there are 3-15 options
- **Single selection** — Choose one value from a list
- **Space-efficient** — More compact than radio buttons

## Basic Usage

<ComponentPreview client:load>
  <SelectBasicDemo />
</ComponentPreview>

```jsx
<Select
  label="Baseline Branch"
  options={[
    { value: 'main', label: 'main' },
    { value: 'develop', label: 'develop' },
    { value: 'staging', label: 'staging' },
  ]}
  placeholder="Choose branch"
/>
```

## When to use Combobox instead

Consider [Combobox](/components/combobox) when:
- Users can type to filter/search options
- The list is long (15+ items)
- Users might need to create new options
- Options are fetched dynamically

## Best Practices

<DosAndDonts client:load>
  <Do>Use clear, descriptive labels for each option</Do>
  <Dont>Use technical IDs or codes as option labels</Dont>
</DosAndDonts>

<DosAndDonts client:load>
  <Do>Order options logically (alphabetically, by frequency, or by priority)</Do>
  <Dont>Use random or inconsistent ordering</Dont>
</DosAndDonts>
