# Introduction

Observatory is a React component library built specifically for visual testing dashboards. It provides a consistent, accessible, and beautiful set of components designed for dark-theme interfaces.

## Philosophy

Observatory is built on four core principles that guide every component decision.

<div className="mt-6 mb-8">
  <PhilosophyCardsDemo client:load />
</div>

### Status Colors Are Sacred

In visual testing, colors communicate critical information at a glance. Users learn to scan by color—never break this trust.

  <StatusColorsDemo client:load />
</div>

## Quick Start

Get up and running in three simple steps.

<div className="my-6">
  <QuickStartStepsDemo client:load />
</div>

Then start building:

```jsx
import { Button, Card, Badge } from '@vizzly-testing/observatory';

function BuildCard() {
  return (
    <Card>
      <Badge variant="success">Approved</Badge>
      <h3>Build #1234</h3>
      <Button variant="primary">View Details</Button>
    </Card>
  );
}
```

## What's Included

Observatory provides **30+ components** organized into intuitive categories.

<div className="my-6">
  <ComponentCategoriesDemo client:load />
</div>

Every component is:

- **Accessible** — Full keyboard navigation, ARIA labels, focus management
- **Dark-optimized** — Designed for dark themes with proper contrast ratios
- **Consistent** — Follows the same patterns, spacing, and interaction models
- **Composable** — Works great standalone or combined with other components
